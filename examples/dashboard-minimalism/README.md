# Example: Dashboard Minimalism

A flat, precise aesthetic system for data-product imagery. It intentionally
pushes away from cinematic warmth: neutral light, muted saturation, clean
digital illustration, and a compact dashboard subject.

The system uses the existing v0.1 vocabulary. `composition_framing` and layout
density are expressed through the `subject` text for now, avoiding a Phase 5
schema expansion before the construct proves reusable.
