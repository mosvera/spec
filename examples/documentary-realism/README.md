# Example: Documentary Realism

An observational photographic system. It uses natural light, natural contrast,
and restrained saturation to test whether Mosvera can express an authentic
documentary look without leaning on the warmer cinematic-editorial vocabulary.

The Phase 5 version deliberately avoids adding `texture`,
`depth_of_field`, or `composition_framing` constructs. Those remain Phase 6
candidate vocabulary if multiple examples need them.
